## Icons

### Library

- **Library**: Lucide  
- **Style**: outline  
- **Import**: `import { IconName } from 'lucide-react'`  
- **Default size**: 20px  
- **Compact size**: 16px  
- **Large size**: 24px  
- **Color**: Inherits from `currentColor` (follows parent text color)

### Usage rules

| Context | Size | Style | Example |
|----------|--------|-------|---------|
| Button with text | 16–20px | Same as DS default | `<Button><Icon /> Label</Button>` |
| Icon-only button | 20–24px | Same as DS default | `<IconButton><Icon /></IconButton>` |
| Navigation / sidebar | 20px | Same as DS default | Menu items |
| Badge / tag | 14–16px | Same as DS default | Inline indicators |
| Illustration / empty state | 32–48px | Same as DS default | Large decorative icons |

### Anti-patterns

- **NEVER use emoji** (😀, 📊, ✅, etc.) as icon substitutes — emojis vary by OS, ignore DS color/size, and break visual coherence.
- **NEVER mix** outline and filled styles on the same screen unless explicitly defined in the DS.
- **NEVER use ad-hoc inline SVGs** if an equivalent icon exists in the chosen library.
